Инструкция для использования конструктора шаблонов

  1. Нажмите "Создать шаблон" для создания нового шаблона документа
  2. Добавьте необходимые поля с помощью кнопки "+ Добавить поле"
  3. Сохраните шаблон под уникальным именем
  4. В разделе "Мои шаблоны" вы можете редактировать, скачивать или удалять созданные шаблоны
