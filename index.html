paperToDOCX — Веб-Приложение Для Автоматизированного Заполнения Шаблонов Документов

Краткая инструкция

  1. Нажмите "Создать шаблон" для создания нового шаблона документа
  2. Добавьте необходимые поля с помощью кнопки "+ Добавить поле"
  3. Сохраните шаблон под уникальным именем
  4. В разделе "Мои шаблоны" вы можете редактировать, скачивать или удалять созданные шаблоны

Веб-Приложение paperToDOCX написано с использованием библиотек:

docxtemplater — для создания и заполнения шаблонов формата .docx

pizzip — вспомогательная библиотека для работы docxtemplater

==================================================

Также используется сервис Cloudflare

Исключительно как CDN (Content Delivery Network) для быстрой загрузки используемых ранее упомянутых библиотек.
